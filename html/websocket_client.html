class websocket_client.WebSocketClient(uri: str, ssl_context: SSLContext)[source]¶

Bases: object

WebSocketClient (Singleton): Класс WebSocketClient реализует паттерн Singleton, то есть при создании нового экземпляра класса возвращается тот же самый объект. Это гарантирует, что WebSocket соединение будет установлено только один раз, независимо от того, сколько раз вы обращаетесь к этому классу.
